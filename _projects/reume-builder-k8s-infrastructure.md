Kubernetes-based infrastructure for deploying and managing the Resume Builder application suite.  
Utilizes Helm charts for application deployment, Terraform for infrastructure provisioning on GCP, and GitHub Actions for CI/CD automation.
This project ensures a scalable, reliable, and maintainable environment for hosting the Resume Builder services, leveraging modern DevOps practices and cloud technologies.
It enables seamless deployment, monitoring, and management of the Resume Builder components, ensuring high availability and performance for end-users.
